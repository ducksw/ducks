== win/loser LigaBulls ==

La mecánica de la liga bulls es esta:

win = 3 points + 15 elo
loser = no points + no elo o (se le baja el elo)

*** La mecánica de los rangos sera así ***

RANK LIST = bronze - silver - oro - platinum - diamond - champion 

ELO Progression and ranks

[ 300  ] --> BRONZE
[ 1000 ] --> SILVER
[ 1500 ] --> ORO
[ 2000 ] --> PLATINUM
[ 2500 ] --> DIAMOND
[ 3200 ] --> CHAMPIONS

**** LIGA BULLS ****

🥇 MEJOR ELO      → [Jugador]    - ELO
🥇 MÁS WINS       → [Jugador]    - WINS
🥇 MÁS DAMAGE     → [Jugador]    - DAMAGE
🥇 MÁS KILLS      → [Jugador]    - INSTANT KILLS

== Player attributes ==

Los jugadores tendran estos atributos.

EXAMPLE = {
  steamId: "76561199093444412",
  displayName: "ducks",
  avatar: "https://avatars.cloudflare.steamstatic.com/ab3a2cf0e52113f3d428d5b2deb80ab67838293e_full.jpg",
  profileurl: "https://steamcommunity.com/id/TheDuckxd/",
  match: "6",
  elo: "365",
  points: "15",
  damage: "13775",
  kills: "0",
  win: "5",
  losser: "1",
  draw: "0",
  rank: "Bronze",
  lastMatches: ["N", "N", "N", "N", "N"],
  isAdmin: true,
  clasificado: false,
}

lastMatches:
    * W = win(ganador)
    * L = loser(perdedor)
    * D = draw(empate)
    * N = null(no ha jugado)
